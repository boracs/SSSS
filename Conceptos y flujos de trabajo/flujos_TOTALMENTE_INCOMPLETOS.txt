                │
                ▼
[ CAPA 3: CAPA DE NEGOCIO (ChatbotAgentService) ]
           │

           │

[ GOOGLE AI SERVICE (Gemini API HTTP client) ] ──► Procesamiento NLU
           │
           ▼
[ EVALUACIÓN POST-RESPUESTA (ChatbotAgentService) ]
           │
           ├─── ¿La respuesta cumple el Guardrail de Certeza Absoluta?
           │    │
           │    ├─── [SÍ] ──► Retorna respuesta limpia en texto plano (Zero XSS)
           │    │             │
           │    │             ▼
           │    │       [ FLUJO DE PERSISTENCIA EXITOSO ]

           │    │
           │    └─── [NO / ALUCINACIÓN DETECTADA] ("No estoy seguro...", "Lo siento, no tengo esa información")

           │    4. Despacha evento interno en el servidor.
           │          │
           │          ▼
